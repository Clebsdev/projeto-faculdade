Manual de Instruções - Sistema SIC

Bem-vindo ao Sistema SIC, uma plataforma dedicada à troca eficiente de informações entre escolas e o serviço de transporte público. Este manual fornecerá orientações passo a passo para escolas e alunos utilizarem o sistema com facilidade.

1. Registro no Sistema:
1.1 Escola:

Acesse o site do Sistema GVBUS. Clique em "Registrar" e preencha as informações necessárias. Após o registro, você receberá credenciais de login.

1.2 Aluno:

Acesse o site do Sistema GVBUS. Selecione "Cadastro de Aluno" e preencha o formulário com os dados solicitados. Após o registro, você receberá credenciais de login.

2. Troca de Informações:
2.1 Escola:

Faça login usando suas credenciais. Complete o formulário online fornecendo os dados dos alunos para a validação do cartão de transporte. O Sistema GVBUS verificará a consistência dos dados e aprovará a liberação do cartão.

2.2 Aluno:

Faça login usando suas credenciais. Preencha o formulário de solicitação do cartão de transporte com seus dados pessoais. Após a submissão, aguarde a aprovação do sistema.

3. Segurança e Privacidade:

O Sistema GVBUS prioriza a segurança e privacidade dos dados. As informações fornecidas são comparadas para garantir a precisão antes da aprovação do cartão. Dados pessoais são tratados com confidencialidade, seguindo as diretrizes de segurança.

4. Busca e Liberação de Linhas de Transporte:

Após a aprovação, o sistema realizará automaticamente uma busca na GVBUS para liberar as linhas de transporte adequadas com base no endereço informado pela escola e pelo aluno.

5. Suporte Técnico:

Para quaisquer dúvidas, em nossa aba dúvidas temos um F.A.Q com as possiveis dúvidas.

Lembre-se, o Sistema SIC está aqui para simplificar o processo de aprovação de cartões de transporte para os alunos. Siga estas instruções para garantir uma experiência eficiente e sem complicações. Obrigado por utilizar o Sistema SIC!
